Real-Time Privacy Manager

Description

Real-Time Privacy Manager is a Google Extension that checks web forms you interact with. The extension extracts the labels and placeholders of each input field you add input to, and gives you a sensitivity analysis with respect to the disclosure of your personal information.

For example:

The extension will extract "This is a label:" and "This is a placeholder" and give you the sensitivity fields of each field.

There are three sensitivity levels:

  • Level 1: Personal information that does not uniquely identify you (first name, date of birth, password...).
  • Level 2: Personal information that uniquely identifies you (email, phone number, passport number...).
  • Level 3: GDPR Special Category (racial or ethnic origin, political opinions, religious or philosophical beliefs...). More information about this category can be found here.

Each sensitivity level has a specific colour assigned to it:

  • Level 1: Green
  • Level 2: Yellow
  • Level 3: Red

When you interact forms, the icon of the extension will change colour in your Google Chrome toolbar. The colour of the icon will be equivalent to the most sensitive field of the form you are filling in.

Installation

To install the extension, please first ensure you have Google Chrome installed on your computer.
You can download Google Chrome here.

If you already have Google Chrome installed, please download the .zip file of the extension using the button below.
Once the archive has been downloaded, extract the files into an empty folder of your choosing.

Then, start Google Chrome and navigate to chrome://extensions.
There, on the top right corner, toggle the developer mode to on.
Afterwards, click on "Load unpacked" and select the folder where you extracted the files.
Finally, please click the puzzle button in your Google Chrome toolbar, scroll towards "Real-Time Privacy Manager", and click on the pin icon.
The icon of the extension should appear in your toolbar.
You are now ready to go.
